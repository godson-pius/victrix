# Victrix

Victrix is a modern digital experience for a visionary organization focused on family health, nutrition, community transformation, and African progress. The website highlights the mission, solution model, team, challenge stories, gallery, and contact information in a polished and immersive format.

## About the project

This website was built with Next.js, React, TypeScript, and Tailwind CSS to deliver a clean, fast, and responsive experience across desktop and mobile devices.

## Key sections

- Home: A premium landing experience for the Victrix brand
- About: Mission and purpose overview
- Solution: A clear presentation of the initiative’s approach
- Team: Profiles and leadership highlights
- Challenges: Story-driven challenge documentation with practical insights
- Gallery: Visual showcase of the work and community impact
- Contact: Ways to get in touch with the organization

## Tech stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- ESLint

## Getting started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Then open http://localhost:3000 in your browser to view the site.
